# ✿ quizzes

a collection of small personality quizzes, made for fun and hosted on github pages.
each quiz is a single self-contained html file — no frameworks, no build steps, just open and play.

---

## live quizzes

| quiz | description | link |
|------|-------------|------|
| if you were a lily... | find out which lily matches your personality | [play →](https://envyforyou.github.io/quiz/lily/) |

> more coming soon :3

---

## structure

```
quiz/
├── README.md
├── LICENSE
├── lily/
│   └── index.html
├── another-quiz/
│   └── index.html
└── ...
```

each quiz lives in its own folder with an `index.html` inside.

---

## adding a new quiz

1. create a new folder with a short name eg. `my-quiz/`
2. put your quiz inside as `index.html`
3. add a row to the table above with the name, description, and link
4. push — github pages will pick it up at `envyforyou.github.io/quiz/my-quiz/`
